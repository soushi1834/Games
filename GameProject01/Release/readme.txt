Shooting Game 説明書

1,操作方法
	Zキー：弾を発射、決定
	Xキー：ボム
	上下左右キー：移動、選択
	Shiftキー：低速移動(押しっぱなし)

2,キャラクター
	青い丸：プレイヤーです。
	黄色い球：プレイヤーの撃った弾です。自分で当たっても死にません。
	赤い丸：敵です。倒すとアイテムを落とします。あたると一機減ります。
	緑の弾：敵の撃った弾です。あたると一機減ります。
	ピンクの大玉：ボスです。倒すとクリアです。あたると一機減ります。
	紫の四角：アイテムです。とるとパワーがたまります。

3,ステータス
	Player：残機です。0になるとゲームオーバーです。
	Bomb：ボムの使える回数です。使うと画面内の敵や敵の弾を消し飛ばします。
	Power：値が高くなると撃つ弾が多くなります。
